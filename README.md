# Miraku

> Track every story.

Miraku is a modern full-stack anime tracking web application designed to help users organize and manage their anime collection through a clean, intuitive, and responsive interface.

## Overview

The goal of Miraku is to provide a seamless experience for tracking anime, organizing watchlists, searching by genres and categories, and viewing personal statistics—all while maintaining a modern UI/UX across desktop and mobile devices.

This project is being developed incrementally following professional software engineering practices, with a strong focus on clean architecture, scalability, and maintainability.

## Planned Features

- User authentication
- Personal anime library
- Watching, Completed, Plan to Watch, Dropped, and On Hold categories
- Multiple genres per anime
- Custom user-defined genres
- Search and filtering
- Ratings and personal notes
- Statistics dashboard
- User profile and account settings
- Responsive design for desktop and mobile
- Dark and light themes
