# Miraku Project Roadmap

## Vision

Create a modern, responsive, and user-friendly anime tracking platform that allows users to organize, manage, and explore their anime collection with ease.

Miraku aims to combine clean design, powerful organization tools, and a seamless user experience while following modern software engineering practices.

---

## Objectives

- Build a scalable full-stack web application.
- Learn modern frontend and backend development.
- Develop a responsive interface for desktop and mobile.
- Practice clean architecture and maintainable code.
- Create a portfolio-quality project.

---

## Target Audience

- Anime enthusiasts
- Casual viewers
- Students learning web development
- Users looking for a clean anime tracking experience

---

### Search & Filter
